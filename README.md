# typizy
A web-based typing speed and accuracy challenge platform.

Typizy helps users improve typing speed and accuracy by providing
randomized multi-line paragraphs fetched from an external API.
It tracks real-time performance including WPM, mistakes, and accuracy,
and offers a clean, animated, and user-friendly experience.
